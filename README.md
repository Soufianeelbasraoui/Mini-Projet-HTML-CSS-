# Calm Landing Page — Mini Projet

**Description**
Reproduction fidèle d'une landing page (maquette Calm) en HTML et CSS uniquement. La page est responsive (desktop + mobile).


**Fichiers**
- `index.html`
- `style.css`
- `images/` (placer les images utilisées)


**Technologies**
- HTML
- CSS
**Instructions**
1. Cloner le repository
2. Placer les images dans le dossier `images/` (noms d'exemple: `hero.png`, `img1.png`, `img2.png`,`logo.png`).
3. Ouvrir `index.html` dans un navigateur.
